# Rails TDD demo

## Build Setup

### Prerequisites

NodeJS and MongoDB setup.
then run the commands below

``` bash
# install dependencies
$ bundler

# serve at localhost:3000
$ bin/rails s
```
and navigate at localhost:8080

### Running the tests

``` bash
# run unit tests
$ bundler exec rspec spec/
```
